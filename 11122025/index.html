was ist eine prüfsumme?

die überprüfung der prüfsumme (checksum/hash) stellt sicher, dass eine heruntergeladene datei vollständig und unverändert ist

bei sha265 hat der hash immer 64 zeichen

die prüfsumme überprüfen

auf dem windows rechner

powershell - Get-FileHash "C:\Pfad\zur\Datei.zip" -Algorithm SHA256

auf dem mac und dem linux rechner

Terminal - shasum -a 256 /Pfad/zur/Datei.zip

für die android geräte gibt es eine app namens hash droid

eine paar sciebo finger übungen

  • einen ordner anlegen
  • eine datei in den ordner hochladen
  • den ordner mit eurem linken nachbarn teilen
  • den ordner so einstellen, dass der nachbar auch eine datei hochladen darf
  • der nachbar nimmt eure datei raus und legt eine datei in den ordner
